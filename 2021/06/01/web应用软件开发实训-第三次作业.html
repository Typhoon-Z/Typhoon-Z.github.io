四、总结

本次实验学习了Git的使用，跟随learngit讲义一步步来操作完成。创建了readme.txt后我就想把它添加到远程仓库上去，但是我一直提交不上去，输入git push -u origin master后显示connection time out。我试了五六七八次都是这样，按照在网上找的资料改了还是不行，关掉又打开也还是不行，用HTTP也不行。哭了。。。实在是改不动了，我去问了同学，她用HTTP帮我push上去了，我发现主要是我没有看后面的讲义，后面中有个操作是 git remote rm origin,是解除本地与远程的绑定关系，之后在使用HTTP git remote add origin…，然后就可以push上去了。

在剩下的操作过程中，我也出了很多小错误，看漏了一些东西，就出现了bug，还要重新操作，不过没有啥大问题。

我觉得之后还要多操作几遍learngit，加深印象，巩固知识。
